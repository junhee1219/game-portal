## 내가 임의로 결정한 것 (후보고)

| 결정 | 이유 |
|---|---|
| repo public | 게임이 이미 github.io에 공개 + 서버 clone 인증 불필요. 비공개 원하시면 settings에서 한 번에 전환 가능 |

| 계측 = 서빙 주입 | 게임 파일 무수정 원칙 유지. 재동기화해도 계측 안 깨짐 |
| 게임 sw.js 무력화 | 서비스워커 캐시가 업데이트를 가리는 사고 방지. 서빙에서만 대체, 원본 그대로 |
