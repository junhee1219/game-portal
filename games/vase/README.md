# 🧪 물병 정렬 (Vase)

알록달록 물을 옮겨 담아 **한 병에는 한 가지 색만** 남기는 퍼즐 게임.

**플레이:** https://junhee1219.github.io/vase/

## 규칙

- 병을 탭해서 선택 → 다른 병을 탭하면 물을 붓는다
- 같은 색 위에만 부을 수 있다 (빈 병은 어디든 OK)
- 모든 색을 각자의 병에 모으면 클리어

## 특징

- 🌊 출렁이는 액체, 물줄기, 기포, 스플래시 — 전부 캔버스 실시간 렌더링 (8색: 파랑·빨강·보라·노랑·초록·검정·흰색·주황)
- 🌌 레벨마다 바뀌는 배경 테마 (8종 순환)
- 🔊 효과음 전부 Web Audio 실시간 합성 (외부 에셋 0개)
- 📳 모바일 햅틱 진동
- ⭐ 별점: 솔버(봇)가 찾은 풀이 길이가 기준 — 봇과 같거나 더 적게 움직이면 별 3개
- 🧠 모든 판은 생성 시 솔버 검증 → 풀 수 있는 판만 출제
- 📈 난이도 곡선: 빈 병이 줄고(4→0) → 더블 컬러로 판이 커진다 (병 최대 16개)
- 🍾 한 색으로 꽉 채운 병은 코르크 마개로 봉인
- 🗺 레벨 선택: 깬 레벨은 언제든 다시 도전 (Lv 버튼)
- 💧 물친구 진화: 최고 레벨에 따라 💧→🫧→🐠→🐙→🐬→🦈→🐳→🐉
- ⏱ 클리어 시 소요 시간 + 이동 수 표시
- 💡 힌트는 솔버가 실제 풀이의 첫 수를 알려줌
- 🏆 레벨/별/최소 이동 기록 저장 (localStorage)
- 📱 PWA — 홈 화면에 설치하면 오프라인에서도 플레이 가능
- 🍩 재밌었다면 클리어 화면에서 개발자 간식 사주기
- 🤫 숨겨진 기능이 어딘가에... 👀

## 구조

```
index.html   마크업 + 스타일
core.js      순수 게임 로직 + 솔버 (node 테스트 공용)
game.js      UI / 애니메이션 / 파티클 / 진행 저장
audio.js     Web Audio 효과음 합성
test.js      로직 테스트 (node test.js)
sw.js        서비스 워커 (배포 시 CACHE 버전 올릴 것)
```

## 개발

```bash
node test.js                  # 로직 테스트
python3 -m http.server 8123   # 로컬 실행 → http://localhost:8123
```
